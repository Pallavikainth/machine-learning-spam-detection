# 📩 SMS Spam Detection using Machine Learning

## 📌 Overview

This project uses Machine Learning and Natural Language Processing (NLP) techniques to classify SMS messages as **Spam** or **Ham (Legitimate)**.

The project includes data preprocessing, feature extraction using TF-IDF, model training, evaluation, and performance comparison.

---

## 🎯 Objective

To build a machine learning model capable of accurately identifying spam SMS messages and reducing unwanted communication.

---

## 🛠 Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

---

## 🤖 Machine Learning Models

- Naive Bayes
- Logistic Regression
- Support Vector Machine (SVM)

---

## 📊 Dataset

- SMS Spam Collection Dataset
- Messages classified as:
  - Spam
  - Ham (Legitimate)
